MikroTik Backup Automation Script

A simple MikroTik RouterOS automation script that creates configuration backups and sends them via email for off-device storage and disaster recovery.

Features

- Exports router configuration
- Creates a binary RouterOS backup
- Sends backups automatically via email
- Generates audit logs
- Supports scheduled execution using RouterOS Scheduler

Use Case

This script was developed to automate configuration backups for MikroTik routers, reducing the risk of configuration loss and improving recovery time during network incidents.

Technologies

- MikroTik RouterOS
- RouterOS Scripting
- SMTP Email Integration

Example Workflow

1. Export current router configuration
2. Create RouterOS backup file
3. Email backup to administrator
4. Log successful execution
